# Módulo CRM

Esta carpeta documenta el CRM operativo del proyecto Agencia Pericial.

## Finalidad

Centralizar la información de cada expediente desde el lead inicial hasta el cierre.

## Qué cubre el CRM

- captación de leads
- cualificación
- seguimiento
- timeline de eventos
- tareas
- documentación
- presupuestos y facturación
- estado final del caso

## Qué no cubre todavía

- multiusuario avanzado
- autenticación robusta
- base de datos relacional
- portal de cliente
- sincronización bidireccional con WhatsApp
