# 09_LOGS

Registro de interacciones con clientes y actas de reuniones internas.

## Estructura

```
09_LOGS/
├─ README.md              # Este archivo
└─ templates/
   ├─ template_llamada.md  # Plantilla para registrar llamadas y videollamadas
   └─ template_reunion.md  # Plantilla para actas de reuniones
```

## Uso

- Crea un nuevo archivo en `09_LOGS/` siguiendo el nombre: `YYYY-MM-DD_tipo_caseid.md`
- Usa las plantillas de `templates/` como base
- Completa todos los campos obligatorios antes de cerrar el log

## Tipos de log

| Tipo | Plantilla | Descripción |
|------|-----------|-------------|
| Llamada / Videollamada | `template_llamada.md` | Registro de contacto comercial con cliente |
| Reunión interna | `template_reunion.md` | Acta de reunión de equipo o decisiones |
